# JSBox-Scripts
存放一些用于分享的 JSBox 脚本。

## 滚动歌词
此脚本旨在配合 Apple Music 使用，通知中心使用效果极佳。配合脚本任务中的固定任务使用极佳。

这个版本增加了主题更改和多语言歌词的支持。详情见 Readme。
